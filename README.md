## 备注

#### git使用

1. ```shell
   git clone https://github.com/weiweiday/python-study.git # 复制一个项目
   ```

2. ```shell
   git add . # 把文件加入版本控制
   ```

3. ```shell
   git commit -m "my first commit" # 提交代码到本地仓库
   ```

4. ```shell
   git push # 上传本地仓库代码到github
   ```

5. ```shell
   git pull # 拉取线上代码到本地
   ```
